# Instruções de uso

1. abrir o código fonte do projeto
2. Na pasta frontend executar `npm install vite`
3. Na pasta backend executar `npm install express`
4. Caso não esteja instalado, é necessário instalar o MySQL (ou equivalente como mariaDB) no sistema
5. Em ambas as pastas frontend e backend executar `npm run dev`
6. IMPORTANTE: Ao executar o projeto pela primeira vez e fazer cadastro não haverá nenhum usuário administrador, então após fazer o cadastro do primeiro usuário a rota localhost:[porta]/admin deve ser acessada diretamente via URL para desbloquear a conta do usuário cadastrado, caso contrário não será possível fazer login.
